Purpose of the Page

The purpose of this page is to become familiar with linux, vi, and git commands. The lists below serve as an initial guideline for the commands utilized to establish a git repository, open the vi editor after initializing, and push commits to the page.

Linux Commands

ls-
Directory Listing.

cd dir -
Change Directory to dir.

pwd -
Present Working Directory.

mkdir -
Create Directory.


VI Commands

vi filename -
edit filename at line 1.

SHIFT + ":" -
Enter command in vi.

ESC -
Turns off Insert Mode in vi.

x -
Write-out and quit vi.

w -
Write into file in vi.

q -
Quit vi.

wq -
Write-out and quit vi


Git Commands

git add -
Add file to next commit (stage).

git branch -
List branches with * as active.

git branch "[Branch Name]" -
Create a new branch at the current commit.

git checkout "Branch Name"
Leave current branch for "Branch Name"

git checkout -b "[Branch Name]" -
Create new branch.

git init -
Initialize existing directory as a Git repository.

git merge [Branch] -
Merge branch’s history into current branch.

git push [Alias] [Branch] -
Transmit local branch commits to remote repository branch.

git status -
Show modified files in the working directory staged for the next commit.

git commit -m "[Message Desc]" -
Commit staged content as a new commit snapshot.
